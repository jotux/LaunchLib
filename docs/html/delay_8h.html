void Delay ( uint32_t  delay_time)

Delay for selected time.

Continuously check global now variable to see if it has passed the delay_time

Warning
Do NOT call this from an interrupt, you will not like the results
Parameters
[in]delay_timetime to delay
{
uint32_t start_time = g_now;
while(g_now < (start_time + delay_time));
}
void DumbDelay ( uint32_t  delay_time)

Delay for selected time.

Sit in a loop and wait for it to end.

Warning
If you call this in an interrupt it will block interrupts until it completes.
Parameters
[in]delay_timetime to delay
{
volatile uint32_t cnt = 0;
while(cnt++ < delay_time);
}
